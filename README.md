# Applied Machine Learning Analysis on Real World Data

Machine learning projects implemented in R using real-world datasets, covering unsupervised learning, regression, and classification.

---

# Project Overview

| Project | Dataset | Techniques |
|----------|----------|----------|
| Unsupervised Learning | Country Socioeconomic & Health Indicators | PCA, K-Means, Hierarchical Clustering |
| Regression Analysis | WHO Life Expectancy Dataset | Linear, Polynomial, Ridge, Lasso, Random Forest |
| Classification Analysis | Diabetes Risk Dataset | Logistic Regression, Random Forest, K-NN, SVM |

---

# Unsupervised Learning

## Objective

Cluster countries based on socioeconomic and public health indicators to identify groups with similar development characteristics.

## Dataset

Country socioeconomic and health indicators dataset

- 167 countries
- 10 variables

## Techniques

- Exploratory Data Analysis (EDA)
- Data Standardisation
- Principal Component Analysis (PCA)
- K-Means Clustering
- Hierarchical Clustering

---

# Regression Analysis

## Objective

Analyse factors affecting life expectancy and compare the predictive performance of multiple regression models.

## Dataset

WHO Life Expectancy Dataset

- 193 countries
- 22 variables

## Techniques

- Linear Regression
- Polynomial Regression
- Ridge Regression
- Lasso Regression
- Random Forest Regression

---

# Classification Analysis

## Objective

Predict diabetes risk using demographic, lifestyle, and health-related factors.

## Dataset

Diabetes Risk Dataset

- 22 variables

## Techniques

- Logistic Regression
- Random Forest
- K-Nearest Neighbours (K-NN)
- Support Vector Machine (SVM)

## Evaluation Metrics

- Accuracy
- F1-Score
- ROC-AUC
